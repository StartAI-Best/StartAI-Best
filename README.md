## 🚀 项目

### [MediaHub](https://github.com/StartAI-Best/MediaHub) 🔜 即将开源
本地素材库 + AI 标注 + 语义搜索 API。自动给音视频图片打标签，支持自然语言搜索，FastAPI + PySide6 桌面客户端。AI 视频生成管道的核心组件，我更愿意叫它为**本地资源大脑**。

### [PPTBot](https://github.com/StartAI-Best/PPTBot) 🔄 更多内容即将更新
集成手机控制 PPT 播放、互动 AI 智能对答、光标功能等，并支持录音、语音转文字分析。

### [GORemotion](https://github.com/StartAI-Best/GORemotion) 🔄 更多内容即将更新
基于 Remotion 的动态场景 AI 视频生成工具，输入描述自动生成视频脚本和画面，可搭配 MediaHub 使用。

## 💡 核心理念

- **本地优先** — 数据不离开你的机器
- **AI 全链路自动化** — 素材管理 → AI 标注 → 语义搜索 → 视频生成
- **开源精神** — 工具应该被理解和定制，而不是被锁在云端

热爱探索新技术，构建让工作流更智能、更高效的工具。
